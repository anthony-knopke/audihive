# Android Task Manager

## Description
A personal Android application built using Kotlin and Android Studio to manage and play music locally.

## Technologies Used
- Kotlin
- Android Studio
- Android SDK

## Features
- Play and pause music tracks with a persistent queue system
- Display album artwork, song title, artist, and track progress
- Edit song metadata (artist, title, album) via a custom tag editor
- Maintain song queue and last played information across app sessions
- Expandable mini player to full-screen player for additional controls

## What I Learned
- Android app architecture and lifecycle management (Kotlin + Android Studio)
- Implementing media playback using ExoPlayer
- Managing persistent data (queues, last played, song metadata)
- Designing responsive and user-friendly UI with Jetpack Compose
- Debugging complex interactions between UI, player logic, and data storage
